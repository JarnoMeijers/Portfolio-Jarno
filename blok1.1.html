Mijn eerste website

In het eerste blok op Hogeschool Zuyd heb ik geleerd een kleine website te ontwerpen en te bouwen. Dit was mijn eerste kennismaking met HTML en CSS. Ik heb geleerd hoe ik een website kan structureren en vormgeven. Het was een uitdagende, maar leerzame ervaring. Via de volgende link kan de website worden bezocht: De link.

Voor dit project heb ik eerst een stijlonderzoek gedaan. Vervolgens heb ik een prototype gemaakt in Figma, dat ik uiteindelijk heb gecodeerd.
